# alx-system_engineering-devops

## Description

This repository contains solutions and exercises for the ALX Software Engineering project **0x00. Shell, Basics**. The project introduces fundamental Linux command-line concepts, Bash scripting, filesystem navigation, file management, shell commands, links, and essential shell utilities used in modern DevOps and system administration environments.

All scripts are written and tested on **Ubuntu 20.04 LTS** and follow the project's specified coding and formatting requirements.

## Project Objectives

Upon completion of this project, the following concepts are understood and applied:

### Shell Fundamentals
- Understanding the Shell and Bash
- Using shell prompts effectively
- Working with command history
- Understanding shebangs (`#!/bin/bash`)
- Understanding the meaning of RTFM

### Navigation
- Navigating the Linux filesystem
- Understanding working directories
- Using:
  - `pwd`
  - `cd`
  - `ls`
- Understanding:
  - Root directory (`/`)
  - Home directories
  - Hidden files
  - Special directories (`.` and `..`)

### Looking Around
- Inspecting files and directories
- Using:
  - `ls`
  - `less`
  - `file`
- Understanding command options and arguments
- Working with long listing formats

### Links
- Creating and managing links
- Understanding:
  - Symbolic links
  - Hard links
- Differences between symbolic and hard links

### File Manipulation
- Copying files
- Moving files
- Removing files
- Creating directories
- Using wildcards effectively

Commands covered include:

```bash
cp
mv
rm
mkdir
```

### Working with Commands
- Identifying command types
- Accessing documentation
- Using:

```bash
type
which
help
man
```

- Understanding aliases
- Knowing when to use `help` versus `man`

### Reading Documentation
- Understanding man pages
- Navigating documentation efficiently
- Understanding man page sections
- Identifying:
  - User commands
  - System calls
  - Library functions

### Bash Productivity
- Common Bash keyboard shortcuts
- Efficient command-line workflows
- Understanding Long-Term Support (LTS) releases

## Requirements

- Ubuntu 20.04 LTS
- Bash shell
- Allowed editors:
  - vi
  - vim
  - emacs

### Script Requirements

- All scripts begin with:

```bash
#!/bin/bash
```

- All scripts are exactly two lines long.
- All files end with a newline.
- All scripts are executable.
- Backticks, `&&`, `||`, and `;` are not allowed.

Example:

```bash
chmod u+x script_name
```
